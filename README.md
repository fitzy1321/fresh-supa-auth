# Fresh project

Current Versions (cuz this $h!T moves fast):

- Fresh : 1.5.4
- Supabase-js: 2.38.4

## Run App

Copy `.env.example`:

```sh
cp .env.example .env
```

Make a supabase project: https://supabase.com




start the project:

```
deno task start
```

This will watch the project directory and restart as necessary.
